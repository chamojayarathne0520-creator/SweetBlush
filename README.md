
# 🍰 SweetBlush

SweetBlush is a modern and responsive dessert shop website developed as a static web project using HTML, CSS, and JavaScript. The website showcases a variety of delicious desserts, including cakes, puddings, cookies, and tiramisu, with an elegant and user-friendly design.

## 🌐 Live Demo

https://chamojayarathne0520-creator.github.io/SweetBlush/

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Attractive homepage with featured desserts
- Product gallery with images and descriptions
- About Us section
- Contact page
- Smooth navigation
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/chamojayarathne0520-creator/SweetBlush.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
